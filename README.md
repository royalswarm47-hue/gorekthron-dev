## 📥 Installation

1. Download `GoreKthronBeta.jar` from [Releases](https://github.com/tvoj-username/GoreKthron/releases)
2. Install **Fabric Loader** for Minecraft 1.21.4
3. Place JAR in `.minecraft/mods/` folder
4. Launch Minecraft with Fabric profile
